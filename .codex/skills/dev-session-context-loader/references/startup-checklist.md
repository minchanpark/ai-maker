# Startup Checklist

1. Run the loader script from project root.
2. Confirm these files were printed if present:
- `dev/AGENTS.md`
- `dev/AI_INSTRUCTIONS.md`
- `dev/PRD.md`
3. Capture constraints and non-negotiables for this session.
4. Start implementation only after confirming scope and acceptance criteria.

## Reload Conditions

- `dev` 문서가 수정된 경우
- 요구사항이 바뀐 경우
- 작업 범위가 크게 변경된 경우
